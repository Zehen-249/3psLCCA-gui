# 3psLCCA App

Welcome to **3psLCCA**, a desktop application for lifecycle cost and carbon assessment workflows.

## What You Can Do

- Create and manage project-based LCCA studies
- Enter structural, demolition, recycling, maintenance, and traffic data
- Track carbon emission and financial impact components
- Review calculated outputs and visualizations
- Recover checkpointed work and validate project data

## Application Modules

The app is organized into major modules under the GUI:

- Structure and material data
- Carbon emission analysis
- Financial data and WPI support
- Demolition, recycling, and maintenance inputs
- Output charts and reporting views

## Quick Start

1. Set up the Python environment with the project dependencies.
2. Launch the GUI entry point from `src/gui/main.py`.
3. Create a new project and populate module inputs.
4. Save checkpoints and review computed outputs.
